## ⚔️ 城外脸谱兵（3 种 × 多帧 = 60 张）

> 黑脸 / 红脸 = 步兵；白脸 = 骑兵；都有完整套装。

### 通用后缀

| 后缀 | 说明 | 用法 |
|---|---|---|
| `body0~3` | 步兵走路 4 帧（无头身子） | 黑/红 用 |
| `w0~3` | 骑兵 4 帧（马腿动画） | 白脸用 |
| `head` | 头（旧版？） | - |
| `nhead` | 正常头（带笑） | 巡逻时显示 |
| `phead` | 凸眼头（被揍懵） | 点击爆头时显示 |
| `poke0~3` | 爆头序列（鼓胀递减） | 点击 0~0.35s 播放 |
| `pop` | 爆头单图（旧版？） | - |
| `sleep0~3` | 渐闭眼序列 | 夜里黑/红睡觉 |
| `sleepbody` | 睡觉站立身（双腿落地） | 同上 |
| `sleephead` | 睡觉头（旧版？） | - |
