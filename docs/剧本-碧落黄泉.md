# 《碧落黄泉》剧本（选定方案）

> 原创神话·上天入地。当前美术（赵云/许褚/曹字旗）为占位，需按「美术改造清单」逐步替换。
> 一句话：她无病无灾，一夜魂散。他不信命，提枪下黄泉、上碧落——翻遍地府的生死簿，
> 杀穿天庭的南天门，只为向那随手一笔便定人生死的神，讨一个「凭什么」。

---

## 一、世界观（三界）

天地分三界：**人间**、**幽冥（地府）**、**天庭（碧落）**。
人的生死由天庭朱笔录于 **生死簿**，地府照簿勾魂，不容更改——这叫 **天命**。

主角 **云中**，人间一名守关剑客（长枪）。其妻 **昭昭**，某个雨夜无故魂散、
尸身冰冷。云中查遍人间无果，只知一事：**她的名字被人从生死簿上勾了。**

他不认这命——**下黄泉寻魂，上碧落问天。**

## 二、情感内核

**逆天改命 · 不平则反 · 深情**。
他闯三界不为当英雄，只为问一句：凭什么高高在上的神，一笔就能抹掉一条无辜的命？
——上穷碧落下黄泉，只为把她要回来。

## 四、结局（抉择，呼应「逆天」）

真相：昭昭之死并非无故——天庭为镇一场将临大劫，需一「引命」祭天，朱笔点了她。
云中问到了「凭什么」，也被推到选择前：

- **【撕命】** 夺回昭昭的魂，把自己的名字换上生死簿——以命易命，她生，他散。（悲壮）
- **【逆天】** 不认这套规则，连生死簿一并焚了——天命崩、三界乱，人从此自主生死，代价未知。（叛逆·续章钩子）

## 五、基地循环（主城场景包装）

- **望乡台 / 古庙**（原主公府）= 阴阳交界落脚处：领路、供奉、引路老者逐关道破三界规则与昭昭之谜
- **农田/香火** = 攒「阳气/香火」，复一块 = 多一分闯界之力（元进度）
- **招兵** = 收拢黄泉游魂 / 天兵叛将结伴（僚机）
- **出征** = 下一段闯界（地府 / 天庭）
