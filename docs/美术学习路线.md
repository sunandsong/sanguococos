# 美术学习路线（为《碧落黄泉》自制素材）

> 目标不是「成为画师」，而是「能画出游戏能用的角色」。
> 配套剧本见 [剧本-碧落黄泉.md](剧本-碧落黄泉.md)。

## 方向：只学「像素画」

对本项目，像素画是唯一正确答案：
- **门槛最低**：格子作画，不用死磕线条/透视/解剖
- **天生带动画**：走路/攻击就是一帧帧画
- **和现有素材统一**：已下载的敌兵本就是像素风
- **资源最多、容错高**

> 手绘/厚涂/骨骼那套先彻底放下。

## 工具：Aseprite（学一个就够）

- 专为像素+动画，行业标准，约 $20
- 免费替代：Libresprite / Piskel（网页版）/ Pixelorama
- 别碰 Photoshop

## 只学这 5 样（按顺序）

1. **调色板**：先别自配色，去 lospec.com 下现成 palette（地府青灰 / 天庭金）
2. **静态站姿**：画一个 32×48 小人（主角「云中」）
3. **明暗**：两三档色阶加体积（像素画核心）
4. **走路循环**：4 帧走路（分水岭，过了就能动）
5. **攻击帧**：3 帧挥砍

> 学到第 4 步就能做出「能进游戏的角色」。

## 教程去哪学

- **B 站**（最对口）：搜「Aseprite 教程」「像素画 入门」「像素 走路循环」——中文免费海量
- **YouTube**（质量最高）：
  - Saint11（Pedro）——像素技巧图解
  - AdamCYounis · Pixel Art Class——系统课
  - Brandon James Greer / MortMort——角色+动画

## 第一周计划

| 天 | 目标 |
|---|---|
| 1 | 装 Aseprite，跟入门视频画个蘑菇/宝箱，熟悉界面 |
| 2 | lospec 挑 16 色 palette，画静态站姿小人 |
| 3 | 加明暗（亮面/暗面/描边） |
| 4-5 | 画 4 帧走路循环，在 Aseprite 预览 |
| 6-7 | 画攻击 3 帧，导出精灵表 |

**第一个交付物**：会走会砍的「云中」——32×48、几帧、导出 PNG 精灵表 → 交给程序接帧进游戏。

## 两条并行（重要）

1. **边学边做**：学画的同时，玩法框架照常推进（关 2~5 地府逻辑不需要美术，火柴人/占位图先顶）。别等美术学会才做玩法。
2. **保留兜底**：学画是长线（角色动画练顺要几周）。短期画不出的 Boss/大件，继续「买素材 + 切图接入」顶上，不冲突。
