# 第二章 · 方案A 落水潜水（原型）

投井下降的「方案A」分支原型（纯 Canvas，`underwater.html` 双击即玩）。与 [chapter2-descend](../chapter2-descend/) 的「方案B 塌井跳台」并列，是同一口井的另一种走法。

## 玩法
- `A/D` 摆动，`S/▼` 下潜，`空格/▲` 上浮 / 从水中跃出，`J` 砸。
- 从井口纵身**长距离自由下落** → **砸破水面**（水花+涟漪）→ 切**水下浮力阻尼**物理，往深处潜到黄泉/地府入口。
- 分支：下落时靠左可落/跳上**水面上方的石台**；台左侧一**堆石头**堵着通道，按 `J` **砸开**（裂纹+碎石飞溅+震动，5 下打穿）→ 钻进洞离开井。

## 视觉 / 氛围
- 背景用真井壁图（井口大图 + 井壁镜像平铺），水面以下叠**黄泉浊水**（越深越浑，透过浑水可见井壁砖纹）。
- 气泡（落水炸开 / 游动吐泡 / 深处升起）、淤泥浮尘、水面波纹涟漪。
- 石台用真平台素材；石堆目前是程序化占位（待换真"碎石堆"绿幕图）。

## 待办

- 真材质（石/岩浆/沙/水/火）应在 Cocos 里用 .effect shader 实现，不在 Canvas 原型。
- A / B 两段可接：B 跳台下降到井底积水 → 砸破水面 → A 潜行/侧道。
