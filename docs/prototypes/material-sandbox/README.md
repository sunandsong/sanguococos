# 材质 / 颗粒 试验（原型）

两个纯前端原型，双击即玩。用来探索「元素 / 流质材质」的两种技术路线。

## sandbox.html — 颗粒沙盒（falling-sand 元胞自动机）★
选材质（沙 / 水 / 岩浆 / 火 / 木 / 石 / 蒸汽 / 橡皮），按住画面撒下，颗粒各自行为：
- 沙堆积会沉进水；水流淌找平；岩浆黏稠流动、点燃木头、上方冒火。
- 火上窜烧木、遇水熄灭变蒸汽；**水 + 岩浆 → 石头 + 蒸汽**。
- 底部可调笔刷 / 暂停 / 清空。
- 技术：二维数组 + 每帧邻居规则（cellular automata）。进 Cocos 同理：一张数据网格 + 每帧规则 + 画到纹理。适合元素怪 / 流沙岩浆浊水 / 可破坏地形。

## materials-shader.html — 像素材质 shader 试验台
石 / 岩浆 / 沙 / 水 / 火 / 溶解 六格，WebGL 片元着色器实时演算（噪声 + 时间 + 域扭曲 + 像素化 + 限色板）。
- 「溶解」格 = 怪物死亡化掉的效果。
- 技术：GLSL 片元着色器。可用引擎：Cocos `.effect`（你的引擎）/ Unity ShaderLab / Godot gdshader / WebGL·Three.js·Shadertoy。

## 两条路线的区别
- **颗粒(sandbox)**：真物理交互、可破坏、会流动堆积；成本随格子数上升。
- **shader(materials)**：材质"看起来"在流动，不占贴图、无限不重复；但不是真颗粒、不能交互破坏。
- 实战常**两者结合**：地形/机关用颗粒，纯视觉材质用 shader。
