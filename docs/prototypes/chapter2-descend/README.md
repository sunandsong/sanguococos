# 第二章 · 投井下降关（原型）

竖版「投井入冥」下降关的可玩原型（纯 Canvas，`descend.html` 双击即玩）。

## 玩法
- `A/D` 左右移动，`S/▼` 穿过脚下平台继续下坠，`空格` 小跳。
- 主角从井口起步台开始，踩着**贴左右井壁交替**的平台一路往下降到井底。
- 每隔几层出一块**很长的横板**（横跨井筒、一侧留缺口），中途接住玩家，需走到缺口再往下 —— 避免一路直坠到底。

## 视觉 / 氛围（已定稿）
- **视角**：平视（正面平墙），不做假透视。
- **井壁**：厚涂破旧夯土砖，竖向**镜像交替平铺**消除接缝横线。
- **平台**：绿幕生成后自动切图；墙侧末端插进井壁消缝；长板用木梁横向拉伸。
- **氛围**：原地轻浮的浮尘（不随下降下落）＋ 缓慢滴落的水滴；水滴落到平台/井底**溅水花**（水色粒子）。
- **配色锁**：灰石 + 土黄褐 + 暖琥珀光，NO green，NOT scary（与第一章厚涂背景统一）。

## 素材来源
`descend.html` 内嵌了 base64 的真实生成图（井壁中段 + 4 件平台）。原图为绿幕/白底，`build_descend.py` 负责扣图（绿幕键控 / 连通域切件）、镜像平铺、拼装。改素材后重跑脚本即可重新生成 HTML。

## 落地到 Cocos 的对应
- 井壁平铺 → `makeScrollLayer`（竖向 + 镜像消缝）。
- 平台贴墙 → 节点按 side 定位 + 外端插进边缘。
- 浮尘/滴水/水花 → 第一章那套天气贴图层 + 粒子；开 Bloom 更佳。
