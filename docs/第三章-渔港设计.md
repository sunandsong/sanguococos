# 第三章 · 小渔港「空港浪话」设计

> 2026-07-22。主线第 3 关:空城之后、航船之前——整关就是"从陆地一步步走向海"。
> 风格:Q版丑萌(见[美术风格锚](美术风格锚-Q版.md));核心新元素:**水**(游泳正式教学关)。

---

## 一、概念:空了的渔村

和空城同一夜——渔村的人也没了。锅里鱼还炖着,晒鱼架还挂着,空船在浪里晃。
只有防波堤尽头一艘船还亮着灯。玩家从村口一路走到那艘船上,开船出海(接第 4 关)。

## 二、段落结构(从陆到海,目标 10 分钟)

```
① 渔村街 → ② 鱼市巷 → ③ 码头栈道(双层水路) → ④ 沉船(必潜段+船舱探索)
→ ⑤ 灯塔(爬塔) → ⑥ 防波堤(浪头) → ⑦ 巨蟹将军 → 上船转场
```

| 段 | 时长 | 内容 |
|---|---|---|
| ① 渔村街 | ~1.5min | 晒鱼架/渔屋,标准战斗;教螃蟹兵(格挡)、河豚(弹射);街尾第一座神龛 |
| ② 鱼市巷 | ~1.5min | 加深的村区:鱼摊棚顶可跳(上层)/巷子里怪多(下层),空城双路线的复习题 |
| ③ 码头栈道 | ~2min | 双层水路主菜:断板/翘板+水下三层(浅水/深水/水底),水鬼、电鳗、气泡、珍珠 |
| ④ 沉船 | ~1.5min | **必潜段**:船壳堵住水面,只能贴船底钻;进阶:船舱内部小迷宫(黑暗+气泡续命+藏宝) |
| ⑤ 灯塔 | ~1.5min | 检查点+**爬塔段**:塔内竖直平台跳跃(河豚弹射当电梯),塔顶远眺亮灯的船(目标预告) |
| ⑥ 防波堤 | ~1min | 石砖长堤,周期浪头拍岸(数拍子走位),浪里夹金币 |
| ⑦ Boss | ~1.5min | 巨蟹将军挡在跳板前;打翻才放跳板 → 上船开船(接第 4 关) |

> 支线:退潮滩涂(限时开放的低洼区,藏珍珠)+ 船舱宝箱——重玩价值。

## 三、双层水路(核心玩法)

复用井关三套游泳帧(竖游/横游/踩水)+ Breath 氧气套件:

```
上层:栈道木板路 —— 快;断板/翘板/渔网陷阱,考跳跃
下层:水里 —— 掉下去不死!变游泳继续前进;慢,有水鬼,但藏钱
```

- 与空城"屋顶/地面"同一设计语言,但性质反转:空城掉下去=死,渔港掉下去=**换一条路**
- 玩家在这关学会"水=另一种路",为 4/6/7/8 关的水戏打地基

## 四、水下规则(氧气=会走路的计时器)

**垂直三层**:

| 层 | 规则 |
|---|---|
| 水面 | 踩水前进,浪会推人(节奏位移),可换气;暴露(水鬼扑脸) |
| 浅水 | 自由游,主路线层 |
| 深水 | 变暗(光圈缩小),氧气消耗加倍;藏珍珠/金币+最凶的怪 |

**水的动词**(渔港启用前四个,其余留给后面关):

- **水流/暗流**:顺流=免费加速爽段;逆流=贴岩石背后歇脚,一推一挡全是节奏
- **气泡点**:水下检查点,补氧;大气泡=水下 LaunchPad,把人往上顶(一物两用)
- **水草**:碰到缠腿,连点挣脱
- **浪头**(防波堤段):周期拍岸,数拍子走位
- 留给后面:漩涡(吸/甩)、闸门水位互换(城堡用)

**水下战斗规则:刀挥不动**——水下只剩冲刺(滑铲变体),遇敌靠**躲和溜**:
无力感=水下紧张感;第 4 关拿枪后水下能开枪,形成能力翻转的爽点。

## 五、怪(水系精怪,Q版丑萌)

| 怪 | 行为 | 教什么 |
|---|---|---|
| **水鬼** | 水里只露俩眼,潜伏不动,进范围才扑 | 看见它、绕开它(水下潜行) |
| **螃蟹兵** | 横着走,双钳格挡正面刀 | 第一个"有防御"的怪:跳劈/滑铲掏后路 |
| **河豚精** | 鼓成球弹射,堵路;引它撞岩自爆 | 怪当钥匙用;也是弹射装置的活体版 |
| **电鳗**(水下) | 周期放电,整条水道变红线 | 背板节奏过 |

## 六、Boss:巨蟹将军

- **形象**:大螃蟹披断网当披风,一只钳大一只钳小(歪的),壳上插着半支船锚——Q版丑萌锚
- **场地**:防波堤尽头平台,右侧就是海,浪头周期拍上台面(场地机制参战)
- **招式**:
  1. **双钳格挡**:正面刀全弹开——绕后/跳劈才有伤害(承接螃蟹兵的教学)
  2. **横甩钳击**:横向大位移扫击,跳过或滑铲穿
  3. **吐泡封路**:一排泡泡墙,戳破会弹人(用弹射机制逃生)
  4. **浪头联动**:低血量时招浪,和它的横甩形成双重节奏
- **定位**:教会"正面打不动就换位置",给第 5 关黑白无常(双Boss夹击)做操作铺垫

## 七、能力/经济

- 新动词:**游泳**(正式教学)+ **弹射**(浪头/河豚/气泡,LaunchPad 首秀)
- 神龛:渔村街尾放本作第一座小神龛(金币→祝福教学点)
- 珍珠:深水限定收集品,高价金币——氧气换钱的风险回报
