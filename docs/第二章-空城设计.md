# 第二章 · 空城(荒诞诡异版)

> 2026-07-18 定向:章节重排 —— 第一章山林(山君虎爷) → **第二章空城** → 城尾老井一跳 → 第三章井/洞(原第二章整体顺延,转场现成)。
> 视觉基调以用户选定的概念图为锚(art/chapter2-city/概念图,待归档),**怪诞诡异、怪而不恐**。

## 一、剧情钩子

云中出山林进城:城门大开,满城无人——**全城人的名字在同一夜被朱笔勾掉了**。
桌上饭还冒热气,晾衣绳静止不动,整座城的东西都微微朝城尾那口老井倾斜,像被井吸着。
这里是天庭"批量销名"的第一现场;昭昭不是个例。关底拿到剧情道具**勾名册**,指向天庭。

## 二、视觉基调(概念图锚)

- 无朝代无地域的怪诞小镇:歪得不讲理的楼、开在二楼通向虚空的门、断在半空的楼梯、
  融化停摆的大钟、问号形路灯、倒浮在半空的伞;
- 色板:灰紫+淡薄荷的天,琥珀色的无人窗光,低饱和怪调;
- **全城元素微微朝井倾斜**(诡异感 + 目的地指向);
- 怪而不恐:禁恐怖/血腥/漆黑,窗光是暖的,荒是"人刚消失"的荒。

## 五、Boss · 城隍爷(怪诞版)

荒废城隍庙的泥塑城隍,半边剥落见泥胎;一手官印(重砸=地面浮朱红大印)、
一手惊堂木(横扫)。他不坏,只是疯了:守着一城空户口本,不许"活人"离城。
动作槽位对齐虎爷(走8 + 横扫4 + 重砸4),代码零改。

## 六、关底衔接

## 七、机制小料(可选)

- 翻倒板车/路障 = 低障碍,滑铲钻过(滑铲键的实战意义);
- 纸鸢妖在屋檐高度飞,跳劈的靶子。

## 七点五、跑酷玩法(2026-07-18 定向)

**节奏原则**:战斗段与跑酷段交替,不混叠。

**第一层·地面障碍(先做,实体表按 x 布置,同 roadSpawns 套路):**

| 元素 | 玩法 | 怪诞皮 |
|---|---|---|
| 低障碍 | 滑铲钻过,撞上弹开小硬直 | 翻倒板车/歪水平的路灯杆 |
| 高障碍 | 跳跃翻越,挡住走路 | 塌半截的砖墙/家什堆 |
| 裂缝 | 跳跃越过,踩空掉血+拉回边缘(不死) | 裂缝里是星空 |
| 弹跳伞 | 踩上自动超级跳(1.5倍高) | 倒浮的伞,"啵嗡"弹起 |

**第二层·屋顶段**:歪楼梯跳上屋顶,高低平台纯跑酷(无怪),末尾跳劈砸下突袭怪窝;
地形升级为分段高度表(2~3档固定高度,不做自由平台)。

**第三层·斜街冲刺(章节高潮)**:井前最后一条街下坡自动加速,只能跳/滑避障,
一路刹不住冲进井口=章节结尾;途中城隍朱红大印从天盖下当移动障碍(Boss预告)。

**配套**:JumpKit 加二段跳(全章通用);落地顺序:第一层→二段跳→斜街冲刺→屋顶段。
