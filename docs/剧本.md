# 《烽火太行 · 游击队长》剧本

> 原创抗日游击战题材。当前美术(赵云/许褚/曹字旗)为占位，需按下文「美术改造」逐步替换。
> 一句话：1941 年，华北敌后。日军一次次扫荡太行山村，一个失去家人的青年扛起大刀，
> 拉起游击队，护住最后的乡亲，把鬼子一寸寸赶出山。

---

## 一、背景

抗战最艰苦的年代。日军对太行根据地反复「扫荡」，烧光、杀光、抢光。
主角的村子 **黑风口** 在一次扫荡中几乎被屠尽。他从死人堆里爬出来，
拾起一把卷了刃的大刀，加入游击队。

从此以黑风口为根据地，一次次出山伏击、破袭、夜袭，护送乡亲、缴枪扩队，
顺着鬼子的据点线，一路打到盘踞县城的日军大队长 **龟田**。

## 二、人物

| 角色 | 占位素材 | 定位 |
|---|---|---|
| **秦大川**（主角） | zhaoyun | 沉默硬汉，大刀游击队长，一家死于扫荡 |
| **伪军** | 步兵(弱) | 汉奸队，炮灰，一冲就散 |
| **日军步兵** | 盾兵/精英 | 三八大盖，硬，守规矩 |
| **掷弹/机枪手** | 弓手(远程) | 火力压制，逼你近身 |
| **龟田大队长**（Boss） | 许褚 | 挥军刀的日军指挥官，盘踞县城 |
| **翠儿 / 老支书** | 待做 | 村里的人，给补给、传情报、寄托牵挂 |

## 三、情感内核

**保家卫国 + 军民鱼水 + 血债血偿**。
打的每一仗背后都是身后的乡亲；越往后牺牲越重，但根据地也一点点活过来。

## 五、结局（通关）

> 硝烟散尽，龟田伏诛。晨光里，乡亲们从地道、从山洞走出来，
> 重新站到黑风口的土地上。
> 老支书拍拍你肩：「大川，仗还长着呢。」
> 你望向东方——**天，要亮了。**

（留「仗还长」口子，续章可做：平原地道战 / 东北抗联雪原 / 大反攻）

## 六、基地循环的叙事包装（主城场景）

- **村公所/指挥部**（原主公府）= 游击队部：领任务、看情报、供奉牺牲战友
- **农田** = 根据地生产（军民大生产），每复耕一块 = 根据地一分元气（元进度）
- **招兵** = 发动群众参军，扩充游击队（出征带民兵僚机）
- **出征** = 出山打鬼子
- 越往后村子越有生气（人回来了、地绿了）→「守住家园」可视化

## 七、氛围复用（现有系统直接用）

密林/夜/雨/落叶/萤火/扬尘全部可用：
- 夜战关 = 夜袭炮楼（招牌桥段）
- 雨天关 = 雨夜奔袭
- 深入关卡逐渐加 **焦土/断壁/据点炮楼** 背景元素 → 「打进敌占区」的压迫感
